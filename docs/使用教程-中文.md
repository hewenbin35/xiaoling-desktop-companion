# 小澪桌面助手中文使用教程

这份教程写给只想“下载、打开、配置、使用”的用户，不要求你懂 Node.js 或 Electron。

## 1. 下载软件包

打开 GitHub 仓库的 Releases 页面，下载：

```text
小澪桌面助手-0.1.0-win-x64.zip
```

下载后右键解压到一个固定位置，例如：

```text
D:\Apps\小澪桌面助手
```

不要直接在压缩包里双击运行。

## 2. 启动小澪

进入解压后的文件夹，双击：

```text
小澪桌面助手.exe
```

也可以双击：

```text
启动小澪.bat
```

如果 Windows 弹出“未知发布者”提示，选择“更多信息”，再选择“仍要运行”。这是因为当前版本没有代码签名证书。

## 3. 基本操作

- 拖动小澪立绘：移动桌宠位置。
- 单击小澪：打开旁边的小回复框。
- 双击小澪：进入完整聊天界面。
- 连按两下 `Alt`：快速召唤回复框。
- 点击其他网页或应用：小回复框会自动收起。
- 右上角关闭按钮：退出小澪。

小澪会记住上次的位置和服装，下次打开会尽量恢复。

## 4. 配置 AI 接口

双击小澪进入大界面，点击设置按钮，填写：

```text
Base URL
API Key
Model
```

示例：

```text
Base URL: https://api.openai.com/v1
Model: gpt-4o-mini
```

也可以填写其他 OpenAI-compatible 接口。

如果没有填写 API，小澪会使用本地占位回复，功能会比较有限。

## 5. 主动说话设置

设置页里可以调整主动性：

- 高存在感：小澪会经常主动冒泡。
- 普通：主动频率适中。
- 安静：很少主动说话。
- 关闭：不主动说话。

也可以设置“安静时段”，例如晚上 23:30 到早上 08:30。安静时段内，小澪不会主动打扰。

## 6. 桌面识别

“识别桌面主动聊天”默认关闭。

开启后，小澪会按设置间隔观察当前桌面截图，并基于画面主动聊天。她不会逐字复述敏感内容，也不会自动执行桌面操作。

如果你正在处理隐私内容，建议关闭这个开关。

## 7. 长期记忆

小澪的长期记忆保存在本地。默认策略是：

```text
只有你明确说“记住”“别忘”“以后叫我...”时，才自动写入长期记忆。
```

你可以在“记忆”页：

- 查看记忆
- 编辑记忆
- 删除记忆
- 一键清除记忆和聊天记录

如果想完全关闭长期记忆，也可以取消“启用长期记忆”。

## 8. 待办和提醒

你可以在聊天里说：

```text
明天下午三点提醒我交作业
```

也可以在设置页的“待办”里手动添加。

提醒到点后，小澪会用桌宠气泡提醒，不会发系统通知。

## 9. 打开网页或应用

小澪只能打开白名单里的网页或应用。

先在设置页“白名单”里添加：

```text
名字：浏览器
目标：https://www.baidu.com
类型：网页
```

或：

```text
名字：记事本
目标：C:\Windows\System32\notepad.exe
类型：应用
```

之后在聊天里让小澪打开对应项目即可。

## 10. 重新开始

如果你想清空聊天和记忆：

1. 打开设置页。
2. 点击“重新开始：清除记忆和聊天记录”。
3. 按提示再点一次确认。

这个操作不会删除你的 API 设置、服装设置、窗口位置、待办、提醒和白名单。

## 11. 常见问题

### npm start 报找不到 package.json

你不在工程目录。先进入工程目录：

```powershell
L:
cd \gmaeAAAAA
npm start
```

如果你只是使用打包版，不需要运行 `npm start`。

### 双击打不开

先确认你是在解压后的文件夹里运行，不是在 zip 里运行。还可以尝试双击 `启动小澪.bat`。

### 其他应用点不了

新版已经尽量收窄点击区域。若仍遇到遮挡，可以先移动小澪位置，或关闭完整聊天/设置界面，只保留桌宠。

### API 没反应

检查：

- `Base URL` 是否正确。
- `API Key` 是否正确。
- `Model` 是否可用。
- 网络是否能访问对应接口。
